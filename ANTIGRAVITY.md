# 心理療法士同好会 Webサイト プロジェクト憲法 (ANTIGRAVITY.md)

## 1. プロジェクト目的
心理療法士の同好会において、会員の情報管理、知見の共有（エッセー、書評）、イベント告知、および会員間の交流を促進するためのプラットフォームを提供する。

## 2. ターゲット
- 同好会の正会員（情報発信・交流）
- 管理運営者（会員管理・コンテンツ管理）

## 3. MVP（最小構成）の範囲
- **フロントエンド**: ホーム（新着表示）、コンテンツ一覧、記事詳細、コメント表示。
- **管理画面**: 会員登録・編集（ログイン/アウト含む）、記事の投稿・編集・削除。
- **データ構造**: 会員、投稿、カテゴリー、コメントの4要素。

## 4. 技術スタック
- **Frontend**: HTML (Jinaja2 Templates) + TailwindCSS
- **Backend**: Python + Flask
- **Database**: SQLite
- **ORM**: SQLAlchemy (Flask-SQLAlchemy)
- **Deployment**: Render（予定）
